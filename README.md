# 자취생을 위한 요리 요약 웹서비스

## 소개
이 프로젝트는 자취생들이 블로그나 유튜브에서 본 요리 레시피를 쉽게 이해하고 따라할 수 있도록 도와주는 웹서비스입니다.  
복잡하게 설명된 요리법을 간단하게 요약하고, 필요한 재료와 순서를 정리해줍니다.

---

## 주요 기능

- 유튜브/블로그 URL 입력 시 레시피 자동 크롤링
- Gemini API를 활용한 요리 단계 요약
- 초보자도 이해하기 쉬운 JSON 구조로 가공
- GitHub Pages 기반 웹 프론트엔드
- FastAPI 기반 백엔드 API

---

## 기술 스택


- Frontend: HTML, CSS, JavaScript
- Backend: Python, FastAPI
- AI 요약: Google Gemini API
- Data 전처리: KNIME (텍스트 정제 자동화)
